Các phím tắt
        1. Ctrl + F : tìm kiếm trong file hiện tại
        2. Ctrl + R : tìm kiếm và thay thế trong file hiện tại
        3. Ctrl + shift + f : tim kiếm trong toàn bộ project
        4. Ctrl + shift + R : tìm kiếm và thay thế trong toàn bộ project
        5. Shift + shift : tìm kiếm every where trong toàn bộ dự án
        6. Shift + F6 : Rename file
