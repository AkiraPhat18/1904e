Thẻ bảng trong html ( table )

        thẻ table để tạo bảng
        thẻ tr tạo hàng hoặc dòng
        thẻ td để tạo cột
        thẻ thead tạo phần header của bảng
        thẻ tbody tạo ra phần thân body của bảng
        thẻ th đóng vai trò như là 1 thẻ cột nhưng mà chỉ áp dụng cho hàng đầu tiên cho bảng
        để làm tiêu đề cho các cột
